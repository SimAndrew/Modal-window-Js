## Modal window JavaScript

---

After opening a modal window it can be closed by clicking, in three ways:

- On close button
- Outside the modal window
- Esc button

#### Run the app

---

- Select branch main
- Run live server

You can try here: [Modal-Window-Js](https://simandrew.github.io/modal-window-js/)
